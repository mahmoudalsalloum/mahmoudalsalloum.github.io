Automated SSD/HDD Monitoring and Backup System with Python

I developed a Python script that runs in the background to monitor the health of SSDs where the operating system and data are stored. The script checks the SSD status every hour using SMART data and detects potential issues, such as bad sectors or signs of failure. If a risk is identified, it sends a notification and automatically performs a differential backup after an initial full backup. The script is configured to start automatically at system boot, ensuring continuous monitoring and data protection. This project demonstrates my expertise in automation, system monitoring, and backup strategies.
